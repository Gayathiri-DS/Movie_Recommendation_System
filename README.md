# 🎬 Movie Recommendation System

A content-based movie recommendation system built using Machine Learning.  
This app suggests movies similar to a selected movie based on genres, overview, and keywords.


## 🚀 Features
- Recommend top 5 similar movies
- Uses TF-IDF vectorization
- Cosine similarity for matching
- Clean and simple UI using Streamlit / Gradio
- No large `.pkl` files required


## 🧠 How It Works
1. Data preprocessing (handling missing values)
2. Feature engineering using:
   - Overview
   - Genres
   - Keywords
3. Convert text into vectors using TF-IDF
4. Compute similarity using cosine similarity
5. Recommend closest matching movies
